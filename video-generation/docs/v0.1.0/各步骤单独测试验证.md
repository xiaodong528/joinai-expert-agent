**执行步骤**:
```bash
# 1. 生成 filelist
for f in clips/scene-*.mp4; do echo "file '$f'"; done > filelist.txt

# 2. 视频拼接

# 3. 音轨混合（如有 TTS + BGM）
ffmpeg -i concat-raw.mp4 \
  -i audio/tts-combined.wav \
  -i audio/bgm.mp3 \
  -filter_complex "[1:a]volume=1.0[voice];[2:a]volume=0.15[bgm];[voice][bgm]amix=inputs=2:duration=first[aout]" \
  -map 0:v -map "[aout]" -c:v copy \
  videos/concat-raw.mp4
```

## 7. 异步 API 统一轮询模式

Stage 4（Seedance）、Stage 6（MiniMax Music）、Stage 8（可灵 Lip-Sync）均为异步 API，统一采用以下轮询模式：

```python
import time
import requests

def poll_task(api_url, task_id, headers, interval=5, timeout=300):
    """统一异步任务轮询"""
    elapsed = 0
    while elapsed < timeout:
        resp = requests.get(f"{api_url}/tasks/{task_id}", headers=headers)
        status = resp.json().get("status")
        if status == "completed":
            return resp.json()
        if status == "failed":
            raise RuntimeError(f"Task {task_id} failed: {resp.json()}")
        time.sleep(interval)
        elapsed += interval
    raise TimeoutError(f"Task {task_id} timed out after {timeout}s")
```

| API | 提交端点 | 轮询端点 | 典型耗时 | 超时阈值 |
|-----|---------|---------|---------|---------|
| Seedance 1.5 Pro | `POST /video_generation` | `GET /tasks/{id}` | 30s-2min | 5min |
| MiniMax Music | `POST /music_generation` | `GET /music_generation/{id}` | 1-3min | 5min |
| 可灵 Lip-Sync | `POST /videos/lip-sync` | `GET /videos/lip-sync/{id}` | 1-5min | 10min |
